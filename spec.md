Overall layout
The interface is divided into multiple dockable panels that can be rearranged, resized, or hidden. 
At the center is the viewport, where the 3D model (and optionally a 2D texture view) is displayed and edited. [experience....adobe.com]

Top area: Main menu & contextual toolbar

The main menu bar (top) provides access to core functions:

File (create/open projects, export textures)
Edit (undo/redo, settings)
Mode (switch between painting and rendering modes)
Window (show/hide panels) [helpx.adobe.com]


Below or integrated is a contextual toolbar that changes depending on the selected tool or layer, exposing relevant settings and shortcuts. [experience....adobe.com]


Left side: Tools toolbar

The vertical toolbar (usually on the left) contains painting tools:

Brush, eraser, projection, clone, polygon fill, smudge, etc. [helpx.adobe.com]


These tools are used directly in the viewport for texture painting.
Some tools support multiple modes (e.g., physical painting or particle brushes). [experience....adobe.com]
